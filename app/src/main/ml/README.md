# ML Models (Private)

The trained ML models have been moved to a **private repository** to protect intellectual property.

## Accessing the Models

The ML models are stored in the private repository:
- **Repository:** [@tombarreras/faces-ml-models](https://github.com/tombarreras/faces-ml-models)
- **Access:** Contact the repository owner for access

## Setup

To use these models, clone the private repository and update your import paths to reference the models from there.
